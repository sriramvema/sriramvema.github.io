Towards Robust Evaluation of Unlearning in LLMs via Data Transformations

This research explores methods for evaluating machine unlearning in large language models (LLMs) through systematic data transformations. The goal is to understand how effectively LLMs can remove learned information from specific data sources without compromising general performance. The work proposes a robust evaluation pipeline and benchmarks across multiple unlearning scenarios to quantify both erasure fidelity and collateral impact.

Can LLMs Obfuscate Code? A Systematic Analysis of Large Language Models into Assembly Code Obfuscation

This study investigates whether large language models can perform code obfuscation tasks traditionally handled by compiler-level transformations. The research analyzes how different model architectures and prompt strategies influence the generation of obfuscated assembly code, examining the balance between obfuscation strength, runtime efficiency, and interpretability. Findings highlight both the potential and current limitations of LLMs in adversarial software generation contexts.
